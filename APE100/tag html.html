# Tag ความหมาย รูปแบบการใช้งาน
1 < html > เป็นแท็กเริ่มต้นและสิ้นสุดของเอกสาร HTML < html >...< /html >
2 < head > เป็นแท็กที่ใช้กำหนดส่วนที่เป็นชื่อเรื่อง < head >...< /head >
3 < body > เป็นแท็กเริ่มต้นและสิ้นสุดของเนื้อหาที่ปรากฎบนหน้าเว็บไซต์ < body >...< /body >
4 < title > กำหนดข้อความบนไตเติลบาร์ < title >< /title >
5 < h1 > เป็นแท็กสำหรับกำหนดหัวข้อให้เป็นตัวอักษรขนาดใหญ่โดยไม่ต้องบอกขนาดตัวอักษร มีตั้งแต่ < h1 > ถึง < h6 > < h1 >...< /h1 >
6 < p > กำหนดย่อหน้า < p >...< /p >
7 < hr > ขีดเส้นจบบรรทัด < hr >
8 size = ...px noshade="" ใช้ร่วมกับ< hr >เป็นการกำหนดความหนาของเส้นจบบรรทัด < hr size=2px noshade="" >
9 < br > ขึ้นบรรทัดใหม่ < br >
10 < pre > สำหรับการเขียนบทกลอน ซึ่งข้อความที่อยู่บนหน้าเว็บไซต์จะปรากฎเหมือนกับในแท็ก < pre >...< /pre >
11 < b > ข้อความหนา < b >...< /b >
12 < strong > ข้อความหนา < strong >...< /strong >
13 < i > ข้อความเอียง < i >...< /i >
14 < em > ข้อความเอียง < em >...< /em >
15 < ul > เป็นแท็กสำหรับแสดงรายการเป็นลำดับโดยใช้วงกลมทึบ ถ้าเป็นลำดับที่อยู่ในลำดับอีกทีจะแสดงเป็นวงกลมโปร่ง < ul >...< /ul >
16 < ol > เป็นแท็กสำหรับแสดงรายการเป็นลำดับโดยใช้ตัวเลข < ol >...< /ol >
17 < li > เป็นแท็กสำหรับแสดงข้อความใน < ul >และ < ol > < li >...< /li >
18 type = 'i' เป็นแท็กสำหรับแสดงรายการเป็นลำดับโดยใช้ตัว i ใช้ร่วมกับแท็ก < ol > และตัว i จะเพิ่มจำนวนเมื่อมีการเพิ่มบรรทัด < ol type="i" >...< /ol >
19 start = 5 เป็นแท็กสำหรับแสดงรายการเป็นลำดับโดยเริ่มต้นจาก 5 ใช้ร่วมกับแท็ก < ol > < ol start= 5 >...< /ol >
20 < img src="" alt="" width="" height="" > เป็นแท็กสำหรับแสดงรูปภาพ โดยเป็นชื่อไฟล์ที่ถูกเซฟ ตั้งชื่อรูป กำหนดความกว้าง ความสูง ตามลำดับ โดยรูปที่นำมาใช้ควรอยู่ในโฟลเดอร์เดียวกันเพื่อให้ง่ายต่อการแสดงรูปภาพ < img src="ชื่อไฟล์.นามสกุล " alt="ชื่อภาพ" width="ความกว้าง" height="ความสูง" >
21 < img src="URL" > เป็นแท็กสำหรับแสดงรูปภาพจากอินเทอร์เน็ต < img src="https://www.abc.com" >
22 < !--...-- > กำหนดเป็นความคิดเห็น โดยจะไม่ปรากฎบนหน้าเว็บไซต์ < !--....-- >
23 < a > เป็นแท็กกำหนดความเชื่อมโยงไปยังเว็บไซต์ต่างๆ < a href="http:/www.abc.com" >Go to abc< /a >
24 < table > เป็นแท็กสำหรับสร้างตาราง < table >...< /table >
25 < tr > เป็นแท็กสำหรับกำหนดแถวในตาราง < tr >...< /tr >
26 < th > เป็นแท็กสำหรับกำหนดหัวข้อในแถว < th >...< /th >
27 < td > เป็นแท็กสำหรับกำหนดข้อมูลในแถว < td >...< /td >
28 align="center" กำหนดข้อมูลให้อยู่ตรงกลางของตาราง ใช้ร่วมกับแท็ก < tr > < tr align="center" >...< /tr >
29 bgcolor="..." กำหนดสีของพื้นหลังบนตารางใช้ร่วมกับแท็ก < tr > < tr bgcolor="yellow" >...< /tr >
30 style="" ใช้สำหรับตกแต่ง เช่น รูปแบบฟ้อนต์ ขนาดฟ้อนต์ สีพื้นหลัง สีตัวอักษร เป็นต้น โดยใช้ร่วมกับแท็กต่างๆ < body style="font-family: 'Courier New', Courier, monospace;" >...< /body >
31 < div > สร้างกรอบล้อมรอบข้อความ < div >...< /div >
32 < span > กำหนดเปลี่ยนสีข้อความเฉพาะจุด < span >...< /span >
33 < small > กำหนดข้อความให้ตัวเล็กลง < small >...< /small >
34 < mark > กำหนดเน้นข้อความที่ต้องการ < mark >...< /mark >
35 < sub > กำหนดข้อความแบบตัวพ่วงท้าย < sub >...< /sub >
36 < sup > กำหนดข้อความแบบตัวยกระดับ < sup >...< /sup >
37 < form > เป็นแท็กสำหรับกำหนดการสร้าง form < form >...< /form >>
38 < input > เป็นแท็กสำหรับกำหนด input ใน form < input type="..." >
